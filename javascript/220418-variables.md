# TIL 
## 변수

1. `const`
  - 유효범위: 블록 레벨
  - 재할당: 금지
  - 중복선언: 금지
  - 호이스팅: X
  - 전역등록: X

1. `let`
  - 유효범위: 블록 레벨
  - 재할당: 가능
  - 중복선언: 같은 스코프에선 불가능
  - 호이스팅: X, 선언문을 만나야 undefined 할당
  - 전역등록: X

1. `var`
  - 유효범위: 함수 레벨
  - 재할당: 가능
  - 중복선언: 가능
  - 호이스팅: O
  - 전역등록: O

- 유효범위 (Scope)
- 재할당
- 중복 선언
- 호이스팅
- 전역 선언시 전역 객체(`window`)의 속성으로 등록
