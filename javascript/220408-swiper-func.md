## swiper 함수

슬라이드 구성할 때 사용한다.

```
new Swiper ('선택자', 옵션{})

new Swiper ('.class', {
direction: vertical(수직), /* 방향 기본값: horizontal(수평) */
autoplay: true, /* 자동으로 움직일 것인가 */ 
loop: true, /* 계속 순환이 되게 할 것인가 */
slidePerView: 숫자, /* 한번에 보여줄 슬라이드 개수 */  
spaceBetween: 숫자, /* 슬라이드 사이 여백 */
centeredSlides: true, /* 1번 슬라이드가 가운데 보이기 */
```
